# slack-rs

[Slack][slack] Web API interface.

[![License][license-img]][license-url] [![Crates.io][crates-img]][crates-url]

# Usage

Add this to your `Cargo.toml`:
```toml
[dependencies]
slack_api = "0.13.0"
```

and this to your crate root:

```rust
extern crate slack_api;
```
